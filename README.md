# bitcamp-study

## Day 1
- 수업의 방향과, 수료 후 로드맵
- 깃, 깃허브 사용법
  1. 파일 가져오기
    (1) Clone (복제) 
    (2) 복제 이후, 추가된 파일은 Pull(가져오기)
  2. 파일 업로드
    (1) 변경된 내용 백업 (백업 파일 명단 작성)
 	$ git add . 
    (2) 명단에 등록한 파일 백업 (local 저장소에 백업한다)
	$ git commit -m "백업설명"
    (3) 로컬 저장소의 변경 내용을 서버에 업로드
	$git push (username, token 요함)
